# 短网址生成器 - 项目规格文档

## 1. 项目概述

### 项目名称
**短址** (Dwz.me - 短网址生成器)

### 核心功能摘要
一款简洁高效的在线短网址生成工具，用户输入任意长链接即可生成简短易记的短链接，支持一键复制和历史记录功能。

### 目标用户
- 营销人员
- 社交媒体运营者
- 需要分享长链接的用户
- 任何需要简化URL的人

## 3. 功能规格

### 3.1 短链接生成

**输入验证**:
- 检测URL格式合法性
- 支持 http:// 和 https://
- 支持无协议URL自动补全

**生成算法**:
- 基于62进制字符集 (a-z, A-Z, 0-9)
- 使用时间戳 + 随机数确保唯一性
- 生成6位字符短码
- 本地模拟短链接格式: `https://d.me/{code}`

### 3.2 复制功能

- 一键复制到剪贴板
- 使用 Clipboard API
- 复制成功/失败提示
- 复制按钮状态变化反馈

### 3.3 历史记录

**存储结构**:
```javascript
{
  id: timestamp,
  original: "原始长链接",
  short: "生成的短链接",
  created: 创建时间戳,
  clicks: 点击次数(预留)
}
```

**功能**:
- 本地 LocalStorage 存储
- 最多保存50条记录
- 显示创建时间
- 可删除单条记录
- 一键清空全部
- 点击短链接可重新复制

### 3.4 用户交互

**输入框**:
- placeholder 提示文字
- focus 高亮效果
- 实时URL格式校验
- 回车键快捷提交

**按钮**:
- hover 颜色过渡
- active 按下效果
- loading 状态显示
- 禁用状态样式
